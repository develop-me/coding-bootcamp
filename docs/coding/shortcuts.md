For each with give commands as `Mac` / `Windows`.

- **Switch application:** `alt + tab` / `Ctrl + tab`

- **Copy:** `cmd + c` / `Ctrl + c`
- **Paste:** `cmd + v` / `Ctrl + v`
- **Cut:** `cmd + x` / `Ctrl + x`

- **Save file:** `cmd + s` / `Ctrl + s`
- **New file:** `cmd + n` / `Ctrl + n`

- **New tab:** `cmd + t` / `Ctrl + t`
